On the night of 14th August 2011, I opened my blog to find it defaced — 
an "Islamic Revolution" flag had replaced my homepage. Someone had also 
added a prompt to download an executable. Classic.

**What happened?**

After some investigation I found that my WordPress theme files had been 
tampered with. The attacker had modified the theme directly — a common 
vector for WordPress sites running outdated themes or plugins.

**How I fixed it?**

- Restored the original theme files
- Reviewed and updated all WordPress plugins
- Changed all passwords

**Takeaway**

If you run a WordPress site, keep your themes and plugins updated. 
Theme files are a common entry point for attackers — treat them like 
code you own, not a black box.

Back online. Managed to survive. 🙂
